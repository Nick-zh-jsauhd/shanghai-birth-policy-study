# FSE Cost Mapping

This folder contains code that converts first-stage FSE outputs into cost-side inputs for downstream optimization.

## Main script

- `model3_stage2_cost_mapping.py`

## Put code here when it does one of these

- Maps estimated effects to monetary costs
- Builds policy cost tables
- Creates optimization-ready cost scenarios
- Compares cost assumptions across scenarios

## Output convention

Default local outputs should go under `outputs/fse/cost_mapping/`.
