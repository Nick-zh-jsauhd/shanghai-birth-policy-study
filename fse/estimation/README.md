# FSE Estimation

This folder contains first-stage estimation code built on top of the cleaned FSE inputs.

## Main script

- `model3_stage1_fe_estimation.py`

## Put code here when it does one of these

- Fixed-effects estimation
- Robustness checks around the FSE design
- Scenario-effect estimation
- Policy-level effect aggregation
- Estimation-side diagnostic plots or tables

## Output convention

Default local outputs should go under `outputs/fse/estimation/`.
