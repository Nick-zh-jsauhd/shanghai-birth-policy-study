# FSE Preprocessing

This folder contains first-stage data cleaning and reshaping code.

## Main script

- `model3_stage0_data_processing.py`

## Put code here when it does one of these

- Reads raw or lightly processed FSE input data
- Cleans respondent-level or scenario-level records
- Builds grouped analytical samples
- Produces intermediate inputs for `fse/estimation/`

## Output convention

Default local outputs should go under `outputs/fse/preprocessing/`.
